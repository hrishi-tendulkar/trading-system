# Strategy Improvement Goal Prompts

Use these files to start parallel strategy refinement workstreams.

Each file contains a paste-ready `/goal` prompt. The expected outcome is always one of:

- a promoted next version,
- a paper-live or sandbox version with explicit limits,
- or a clear research-only / no-change decision.

Current prompts:

- `01-breakout-confirmation-goal.md`
- `02-sector-confirmed-pullback-continuation-goal.md`
- `03-etf-trend-rotation-goal.md`
- `04-selective-mean-reversion-goal.md`

When adding a new strategy, create:

```text
{NN}-{basis_code}-goal.md
```

and make sure the prompt names the stable `basis_code`, current version, promotion bar, benchmark criteria, required docs, and done condition.
