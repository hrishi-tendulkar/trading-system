Backtest Learnings

What the replay is telling us about the logic

Backtests belong after the weekly board, not before it. Their role is to calibrate trust in the logic that generated the actions above.

The replay still demands humility
The top-two portfolio is only -0.22% vs SPY on average, so the product should present current opportunities as disciplined setups, not as a system that has already proven strong outperformance.
A live setup can still deserve tempered trust
`Constructive pullback continuation` fits the daily-data logic well, but in this tiny replay it only delivered -1.06% with a 27% win rate. That should reduce how confidently the page speaks, even when the chart looks good right now.
Backtests are still technical-only
The current replay does not yet include historical revision data, transcript changes, or point-in-time filing deltas. Strategy pages should say that clearly instead of implying richer event intelligence than the stack actually has.
