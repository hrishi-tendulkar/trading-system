# Trading System Frontend Design Agency Handoff

This folder contains the materials to send to a frontend design agency.

## Recommended Packet

Send these files:

1. `frontend-design-brief.md`
2. `docs/design/information-hierarchy.md`
3. `prototype-reference.md`
4. `agency-cover-note.md`

Also include the rough HTML prototype folder:

`docs/design/prototypes/weekly-decision-redesign/`

Important: the prototype is only a workflow/data reference. It is not the desired final visual design.
