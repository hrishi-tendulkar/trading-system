# Cover Note To Agency

Hi,

We are looking for a frontend UX/design partner to redesign the core pages of Trading System, a single-user weekly trading decision app.

The goal is not to invent new trading logic or new product features. The goal is to take the existing workflow and data objects and create a much clearer, more professional, decision-first user experience.

We are attaching:

- a design brief,
- a rough HTML prototype,
- and notes explaining how to interpret the prototype.

Please treat the HTML prototype as a functional reference only. It shows the page types, navigation paths, and data relationships. It is not the desired visual design, layout, typography, or final UX quality bar.

We expect you to challenge the prototype, simplify the navigation, improve hierarchy, improve language, and create a bar-raising experience for an investor trying to decide what to trade each week.
